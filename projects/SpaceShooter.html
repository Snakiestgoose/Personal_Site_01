Space Shooter

Project Type University Project
Software Used Unity
Languages Used C#
Primary Role(s) Game Programming

Overview

This Space Shooter game is a university project that follows a unity tutorial with modifications. The assets are not my own, however, everything else is.
Through the assignment I created this standard space shooter with moving aliens and asteroid projectiles coming towards the player. The modifications I added includes the main menu, the alien mode that only spawns alien enemies at a different rate that the standard mode, and the force field. The force field activates by the space bar and has a five second timer after coming into contact with an object. There is also added UI to show this process.
